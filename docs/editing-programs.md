# Editing Programs in Contentful

This guide explains how to add, edit, or remove programs shown on the **Programs** page of the Mama Tulia website.

## Finding Programs

1. Log in to [Contentful](https://app.contentful.com)
2. Click **Content** in the top menu
3. Filter by content type: **Program**
4. You'll see all 9 programs listed

## Editing a Program

Click on any program to open it. You'll see these fields:

| Field | What it does | Example |
|-------|-------------|---------|
| **Program Name** | The title shown on the website | "Hospital Visits" |
| **URL Slug** | Used in the web address (don't change unless needed) | "hospital-visits" |
| **Short Summary** | 1-2 sentence overview shown on the page | "We visit hospitals across Uganda..." |
| **Full Description** | Longer text with more detail | The full program description |
| **Program Image** | Photo shown next to the program | Click "Link existing asset" to pick a photo |
| **Category** | Groups programs on the page | "Core Program", "Support Program", etc. |
| **Display Order** | Controls the order programs appear (1 = first) | 1, 2, 3... |
| **Show on Website** | Toggle to hide/show a program | Check = visible |

## Categories

Programs are grouped by category on the website:

- **Core Program** — Main programs (Hospital Visits, Home Visits, etc.)
- **Support Program** — Additional support (Food Relief, Purifier Project)
- **Medical Initiative** — Medical programs (ROP Screening)
- **Stories** — Testimonies and stories

## Adding a New Program

1. Click **Add entry** → **Program**
2. Fill in all required fields (Name, Slug, Summary, Category)
3. Pick a **Category** and set the **Display Order**
4. Optionally add an image and full description
5. Make sure **Show on Website** is checked
6. Click **Publish**

## Changing a Program's Image

1. Open the program entry
2. Scroll to **Program Image**
3. Click the current image to remove it, or click **Link existing asset**
4. Browse the media library and select a photo
5. Click **Publish**

## Hiding a Program

To temporarily hide a program without deleting it:

1. Open the program entry
2. Uncheck **Show on Website**
3. Click **Publish**

The program will no longer appear on the website but the content is preserved.

## Reordering Programs

Change the **Display Order** number:
- Lower numbers appear first (1 = top)
- Programs within the same category are sorted by this number

## Tips

- **Summary** should be 1-2 sentences — this is what people read first
- **Full Description** can be as long as needed — only the first ~300 characters show on the page
- Use landscape photos (wider than tall) for best results
- Changes appear on the website within 1 minute after publishing
